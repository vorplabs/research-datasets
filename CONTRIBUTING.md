# Contributing

This is a research-data mirror, not a general software project. The most useful contributions are source-backed corrections, missing primary sources, reproducibility checks, and clearly scoped schema suggestions.

Before proposing a change, read the canonical page and dataset manifest. Do not submit personal data, confidential material, legal conclusions, bulk-generated rows, or sources that cannot be independently checked.

Use the process in [CORRECTIONS.md](CORRECTIONS.md). Vorp Labs Research may decline changes outside a dataset's stated scope.
